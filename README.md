# Smartium 🩺

A beautiful, modern learning platform for medical students. Built with React + Vite, featuring stunning animations and a professional cyan-blue design.

![Smartium Preview](https://via.placeholder.com/1200x600/0f172a/06b6d4?text=Smartium+-+Master+Medicine+Smarter)

## ✨ Features

- **📚 Lecture Summaries** - Comprehensive, concise summaries of medical lectures
- **🧠 Smart Flashcards** - AI-powered spaced repetition for optimal retention
- **📝 Practice Exams** - Realistic exam simulations with detailed explanations
- **📊 Progress Tracking** - Monitor your learning journey
- **🎯 Performance Analytics** - Insights to improve your study strategy

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
cd Smartium
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5174](http://localhost:5174) in your browser.

## 🛠️ Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations

- **React Router** - Navigation
